# Salesforce Query Tool with Natural Language

This project aims to build a simple tool that allows users to query Salesforce data using natural language and visualize the results in a dashboard.

## Tech Stack

- **Backend:** Python (FastAPI/Streamlit)
- **Salesforce Integration:** Salesforce MCP (Model Context Protocol)
- **Natural Language Processing:** Simple LLM wrapper (e.g., OpenAI or similar)
- **Frontend/Dashboard:** Streamlit (for rapid prototyping)

## Development Plan

1. Set up the environment.
2. Connect to the Salesforce MCP server.
3. Implement NL-to-SOQL translation.
4. Build the Streamlit dashboard.
